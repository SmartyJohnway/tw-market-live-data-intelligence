# M7G Frontend Manual Safe Artifact Load UI

Status: `frontend_manual_safe_artifact_load_ui_defined`

The UI provides a file input, paste JSON area, validate button, load validated button, reset to demo button, validation result panel, loaded artifact manifest/provenance panel, and rejection summary panel.

Loading requires manual operator action only. Rejected artifact never reaches renderer. There is no hidden fetch, no auto refresh, no backend/API/MCP, no live probe, and no URL artifact loading.
