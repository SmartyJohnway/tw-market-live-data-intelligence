M5OP Local Operator Workbench

Normal Mode C workflow: open frontend → review canonical context → review watchlist → plan live observation → explicitly execute bounded observation → review caveats/failures → build conversation context → copy to AI chat.

Local API connection

If API is unavailable, start FastAPI with uvicorn server.main:app --host 127.0.0.1 --port 8000. This workbench makes only local API calls and has no hosted/cloud endpoint fallback.

No trade advice, ordered preference lists, valuation objectives, order-side labels, broker/auth, polling, scheduler, full-market scan, or raw endpoint payload handoff. M5F is canonical but not live; Level 2 observations are not canonical.
